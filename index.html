Dynamic 6-Profile Tax Architecture

Handles all real-world Egyptian & International invoicing tax systems dynamically.

Profile A: Explicit Per-Line Tax

Retains printed per-line VAT amounts directly from vendor table columns without altering paper figures.

Profile B: Tax Exempt / 0% VAT

Detects Freezone, exempt (معفي), and 0% VAT terms automatically, calculating clean subtotal lines.

Profile C: Implied Stated Diff

Infers unprinted per-line tax from stated total differences whenever line subtotal exceeds net unit prices.

Profile D: Proportional Allocation

Splits overall document footer summary tax proportionally across items based on individual line financial weight.

Profile E: Inclusive Decomposition

Decomposes stated prices (السعار شاملة ضريبة 14%) into true pre-tax price & VAT component without double taxation.

Profile F: Exclusive Tax Addition

Calculates 14% exclusive VAT on pre-tax base prices when terms state prices exclude VAT (تضاف ضريبة).
